Implementierung

Zwei Entwicklungsaktivitäaten werden unterstützt:

  • Das physikalische Datenmodell kann aus dem logischen Datenmodell abgeleitet werden. Es können aber auch manuell Tabellen angelegt bzw. geändert werden. Diese manuellen Eingriffe bleiben bei Änderungen durch erneutes Ableiten aus dem logischen Datenmodell möglichst unverändert.
  • Die generische Darstellung einer Scorecard kann festgelegt werden.
